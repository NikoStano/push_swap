# Push_Swap [![nistanoj's 42 push_swap Score](https://badge.nimon.fr/api/v2/cmgjygisn1482501pa9h6l9sg4/project/4394481)](https://github.com/Nimon77/badge42)

Ce dépôt contient mon travail pour le projet push_swap. Le but est d'écrire un programme qui trie une liste d'entiers en utilisant uniquement deux piles et un ensemble d'opérations prédéfini, en minimisant le nombre d'opérations.

## Utilisation

- ./push_swap <liste d'entiers>
  Exemple : ./push_swap 3 2 1 0 6

- Le programme affiche la suite d'opérations pour trier la pile (une instruction par ligne).

## Règles et opérations autorisées

Vous devez trier la pile en utilisant uniquement les opérations autorisées par le sujet push_swap : sa, sb, ss, pa, pb, ra, rb, rr, rra, rrb, rrr. Référez-vous au sujet officiel pour la description exacte de chaque opération.

## Tests et vérification

Pour vérifier le résultat vous pouvez utiliser un programme "checker" (non fourni dans ce dépôt) qui applique les opérations et confirme si la pile est triée.
